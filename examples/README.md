# Integration Examples

This directory contains examples of how to integrate the PRD Extractor into different applications.

## Examples

1. **express-app.js** - Express.js application integration
2. **nextjs-api.js** - Next.js API route integration
3. **python-client.py** - Python client example
4. **react-upload.jsx** - React file upload component
5. **standalone.js** - Standalone script usage

## Running Examples

### Express.js Example
```bash
node examples/express-app.js
```

### Next.js Example
Copy `nextjs-api.js` to your Next.js `pages/api/` directory

### Python Client
```bash
python examples/python-client.py path/to/document.pdf
```

### React Component
Import the component into your React application
